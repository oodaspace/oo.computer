# oo.computer

See https://oo.computer

## To Run:

1. clone this repo `git clone https://github.com/oodaspace/oo.computer.git`
2. `cd oo.computer`
3. `npm install`
4. `npm run start`

## Build instructions

1. clone this repo
2. cd to root dir (the one with package.json in it)
3. `npm install`
4. `npm run dist`
5. cd to dist and run oo.computer
